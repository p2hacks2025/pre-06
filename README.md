## コンセプト
Luminaは、毎日出されるお題をきっかけに、何気ない日常を思い出として残すアプリです。
共有は一瞬、記録はずっと残ることで、日常の中の「キラキラ」に気づける体験を提供します。

## 対象ユーザ
若者
特別でない日常も思い出として残したい一方、日記は続かず、他人の評価を気にせず共有できる場を求めている人にとってぴったりなアプリです。

## 利用の流れ
ユーザーはLuminaを起動すると、その日だけの「お題」が表示されます。
お題は毎朝更新され、特別な準備をしなくても、日常の中で自然に意識できる内容になっています。
ユーザーはお題に合わせて、写真または短い文章を投稿します。
投稿は深く考えなくてもよく、数秒で完了します。
投稿後は、他のユーザーの投稿を見ることができます。
他のユーザーの投稿は毎回ランダムに表示され、24時間後には自動的に消えるため、評価や比較を意識せず、気軽に楽しむことができます。
一方で、自分の投稿はアルバムとして保存され、後からいつでも振り返ることができます。
Luminaを使い続けることで、日常の何気ない瞬間が少しずつ思い出として積み重なっていきます。

## 推しポイント
1,お題があるから、日常に気づきが生まれる
毎日提示されるお題が、ユーザーに新しい視点を与えます。特別な出来事がなくても、「今日の光」「最近うれしかったこと」などのお題を通して、何気ない日常の中に価値を見つけることができます。これは、ただ記録するだけの日記アプリやSNSにはない体験です。

2,比較や承認欲求から距離を置いたSNS
フォロワー関係を持たず、投稿はランダムに表示されます。
そのため、フォロワー数や「いいね」を意識する必要がなく、
安心して自分のペースで使うことができます。
SNS疲れを感じているユーザーにとって、心理的負担の少ない空間を提供します。

3,使い続けるほど価値が増すプロダクト
Luminaは、使い続けることで自分だけのアルバムが完成していきます。後から振り返ったとき、普通の日常が「大切な思い出」になっていることに気づける点が、ユーザーにとっての大きな嬉しさです。
